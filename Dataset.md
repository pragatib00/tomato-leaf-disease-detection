## Dataset Information

The model was trained using the PlantVillage Tomato Leaf Disease Dataset available on Kaggle:

Dataset: Tomato Leaf Disease Dataset
Source Link: https://www.kaggle.com/datasets/kaustubhb999/tomatoleaf

## Dataset Details

10 tomato disease classes (subset of 4 used in this project)
RGB leaf images
Structured into training and validation folders
Balanced class distribution


For this project, the following 4 classes were selected:

- Early Blight
- Healthy
- Late Blight
- Septoria Leaf Spot
